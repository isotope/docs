# Kategorie-Filter

<docrobot_new_in_version version="2.6"><p>Dieses Feature ist neu in Isotope eCommerce 2.6</p></docrobot_new_in_version>

Das Modul `Kategorie-Filter` ermöglicht es Produkte nach Kategorien zu filtern.
